This page documents the currently implemented parts of the universal CSS system. It uses only classes from the shared CSS library without page-specific styling.

Current theme: internal

Current direction: ltr

1. Foundation Layer

The foundation layer includes tokens, base styles, layout rules, direction support, width values, blur values, and theme variables.

  • Tokens: colors, typography, spacing, radius, shadows, widths, and blur.
  • Base: reset, root styles, fonts, and direction rules.
  • Layout: container and grid system.
  • Themes: external and internal semantic color mapping.

2. Theme System

The same components should automatically change color when switching between internal and external themes.
