Theme System

The theme system controls the main visual identity of mngha Universal CSS. Use it to switch the ready styles between external and internal themes without changing component CSS.

View Foundations

Theme Overview

External Theme

Uses the external green brand direction.

Internal Theme

Uses the internal blue brand direction.

Ready Components

Components already read from theme tokens and update automatically.

How to Apply Theme

Apply the theme from the root HTML element using the data-theme attribute.

Apply theme

<html lang="en" dir="ltr" data-theme="external">

<html lang="en" dir="ltr" data-theme="internal">

Theme Usage Rules

Do not declare new colors

Pages and applications should not add custom colors outside the design system.

Use themed styles

Use the ready components and theme color tokens so the UI changes correctly between themes.

Switch from the root only

Change the active theme using data-theme on the html element, not by overriding component styles.
