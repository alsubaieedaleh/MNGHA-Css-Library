Date Picker

Date pickers let users type or select a single date or a date range. The calendar opens directly below its input and includes working month and year menus, navigation, input synchronization, range selection, disabled dates, outside-month dates, keyboard navigation, and accessible state updates.

View Components

Included Features

Single date

Select one date, type a value using DD/MM/YYYY, clear the value, or return to today.

Date range

The first and last dates use the solid brand color. Dates between them use the light brand range background.

Month and year menus

The calendar menus reuse the pagination menu behavior with a wider, scrollable surface and a selected check mark.

Interactive Single Date Picker

The calendar opens as an overlay directly below the input. No helper text is required; use the input placeholder for the expected format.

      Implemented Behavior

      Mouse and touch

      Open from the input, select dates, use month and year menus, navigate months, select today, clear, and close outside.

      Keyboard

      Arrow keys move by day or week. Home and End move within a week. Page Up and Page Down change the month. Enter and Space select. Escape closes.

      Accessible updates

      The script synchronizes aria-expanded, aria-selected, aria-current, aria-disabled, labels, grid names, and roving tabindex.

      Class Reference

      Class Element Purpose
      .mngha-date-picker Container Creates the calendar surface and layout.
      .mngha-date-picker-anchor Container Positions the calendar directly below its input.
      .mngha-date-picker-dropdown Container Positions a month or year menu.
      .mngha-date-picker-menu Listbox Creates the wider scrollable calendar menu.
      .mngha-date-picker-menu-item Button Creates a selectable month or year option.
      .mngha-date-picker-days List Creates the seven-column date grid.
      .mngha-date-picker-day Button Provides the interactive date cell.
      .is-selected Date button Marks one selected date or a range endpoint.
      .is-range-start Date button Marks the solid first date in a range.
      .is-in-range Date button Marks middle dates using the light brand background.
      .is-range-end Date button Marks the solid last date in a range.
      .is-today Date button Marks the current date.
      .is-outside Date button Marks a date outside the visible month.
      .is-disabled Date button Marks an unavailable date.

      Usage Guidance

      Do

      Keep typed input available, use a clear placeholder, preserve keyboard access, provide full date labels, and use disabled state only for dates that cannot be selected.

      Avoid

      Do not mark every range date as is-selected. Use is-selected only on the first and last dates, and use is-in-range for the dates between them.
