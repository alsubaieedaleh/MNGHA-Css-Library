Quick Usage

Import the compiled CSS file

Add the compiled stylesheet once in the page. All components, tokens, themes, layout utilities, and direction support come from this file.

<link rel="stylesheet" href="./../mngha-ui.css">

Use the shared docs header

Documentation pages should load the common header from docs/partials/header.html instead of repeating the full header markup in every page.

<div id="docs-header"></div>

Use the shared docs script

Theme switching, direction switching, active navigation, and header dropdown behavior should live in one shared docs JavaScript file.

<script src="./assets/docs.js"></script>

System Rules

Use tokens

Do not introduce new colors, font sizes, spacing values, or radius values inside pages.

Use components

Build pages from existing components such as buttons, cards, forms, tables, and pagination.

Use templates

Full pages like login, dashboard, and error pages belong in templates, not components.
