Pagination

Pagination helps users move between multiple pages of content. Use it independently or inside a table footer.

View Components

Overview

Numbered navigation

Page items allow direct navigation to a specific page.

Previous and next

Arrow controls move one page backward or forward.

More pages

The dropdown gives access to page numbers that are not displayed.

Standalone Pagination

Use this pattern below lists, cards, search results, and other paged content that is not presented in a table.

States

Use these classes and attributes to represent pagination states.

State Class or attribute Usage
Default page .mngha-pagination-item A selectable page item.
Current page .is-active + aria-current="page" Marks the page currently being displayed.
Disabled control disabled Use when previous or next navigation is unavailable.
Open menu .mngha-pagination-dropdown.is-open Displays the more-pages menu.
Selected menu item .mngha-pagination-menu-item.is-selected Marks the selected page in the menu.

Class Reference

These are the public pagination classes used by the examples.

Class Purpose
.mngha-pagination-wrapper Provides horizontal overflow protection.
.mngha-pagination The main pagination navigation container.
.mngha-pagination-item A numbered page button.
.mngha-pagination-arrow A previous or next page control.
.mngha-pagination-dropdown Contains the more-pages trigger and menu.
.mngha-pagination-more Opens the hidden page menu.
.mngha-pagination-menu The list of additional page options.
.mngha-pagination-menu-item A selectable page inside the menu.

Usage Guidelines

Mark one current page

Apply is-active and aria-current="page" to one item only.

Disable unavailable arrows

Disable the previous control on the first page and the next control on the last page.

Keep labels accessible

Give the navigation, arrows, and more-pages control clear accessible labels.

Dropdown JavaScript

Numbered pagination can be connected to any application logic. This script only manages the more-pages menu.

JavaScript

Pagination dropdown JavaScript

function initPaginationDropdowns() {
  const dropdowns = document.querySelectorAll(
    ".mngha-pagination-dropdown"
  );

  dropdowns.forEach(function (dropdown) {
    const trigger = dropdown.querySelector(
      ".mngha-pagination-more"
    );

    const items = dropdown.querySelectorAll(
      ".mngha-pagination-menu-item"
    );

    if (!trigger) return;

    trigger.addEventListener("click", function (event) {
      event.stopPropagation();

      const shouldOpen = !dropdown.classList.contains("is-open");

      closePaginationDropdowns();

      if (shouldOpen) {
        dropdown.classList.add("is-open");
        trigger.setAttribute("aria-expanded", "true");
      }
    });

    items.forEach(function (item) {
      item.addEventListener("click", function () {
        items.forEach(function (menuItem) {
          menuItem.classList.remove("is-selected");
        });

        item.classList.add("is-selected");

        closePaginationDropdowns();
        trigger.focus();
      });
    });
  });

  document.addEventListener("click", closePaginationDropdowns);

  document.addEventListener("keydown", function (event) {
    if (event.key === "Escape") {
      closePaginationDropdowns();
    }
  });
}

function closePaginationDropdowns() {
  const openDropdowns = document.querySelectorAll(
    ".mngha-pagination-dropdown.is-open"
  );

  openDropdowns.forEach(function (dropdown) {
    dropdown.classList.remove("is-open");

    const trigger = dropdown.querySelector(
      ".mngha-pagination-more"
    );

    if (trigger) {
      trigger.setAttribute("aria-expanded", "false");
    }
  });
}

initPaginationDropdowns();
